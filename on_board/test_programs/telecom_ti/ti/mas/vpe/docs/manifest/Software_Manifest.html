Manifest for VPE Component Version 3.6.0.0

Voice Processing Elements Component. Contains the High Level Compensation (HLC), PCM Encoder/Decoder (PCM), Sample Frequency Converter (SFC), Signal Limiter (SLM), Speech and Noise Level Estimates (SNL), Simple Voice activity Detection (SVD) and Adaptive Spectral Noise Reductin (ASNR) modules.
